Часто задаваемые вопросы о Karing

Приложение Karing

Бесплатно ли приложение Karing? Сколько оно стоит?

  • Полностью бесплатно
  • Для пользователей Apple: все аккаунты Apple вне материкового Китая могут скачать приложение
  • Для Windows-версии, пожалуйста, перейдите на Github для скачивания (возможно потребуется открыть прокси для загрузки)

Какие платформы поддерживаются?

  • Уже поддерживаются: IOS, MacOS, TvOS, Windows, Android
  • В разработке: Linux

Какие типы конфигураций подписки поддерживаются?

  • Clash, V2ray (поддерживает пакетную обработку), Stash, Karing, Sing-box, Shadowsocks, Sub, Github

Какие протоколы поддерживаются?

  • Shadowsocks, ShadowsocksR, VMess, Vless, Trojan, Tuic, Socks, Http, Hysteria, Hysteria2, Wireguard, AnyTLS, Mieru и другие протоколы

Какую версию следует скачивать пользователям Apple?

  • Если вы хотите стабильного использования, рекомендуется скачать версию из AppStore
  • Если вы хотите попробовать новые функции, можно скачать бета-версию из TestFlight

Как использовать приложение?

  • Для новых пользователей, которые хотят быстро начать работу, рекомендуется обратиться к руководству быстрого запуска на этом сайте

Как обновить приложение?

  • Для пользователей Apple: используйте системное обновление
  • Для пользователей Windows/Android: Karing будет подсказывать после автоматического обновления, просто скачайте или перезапустите приложение согласно подсказке

Это безопасно?

  • Приложения проходят проверку Apple перед размещением в AppStore, и версия Apple может быть скачана только из AppStore или TestFlight (пожалуйста, не скачивайте приложения из неофициальных источников)
  • Мы также соблюдаем те же правила безопасности и конфиденциальности, что и платформы Apple, для обеспечения того, чтобы ваша информация не была раскрыта
  • Пользователям не-Apple платформ рекомендуется скачивать с Github
  • Официальные источники загрузки Karing: только AppStore и GitHub, пожалуйста, не скачивайте и не устанавливайте из других источников
  • Karing не размещён в Google Play, пожалуйста, не скачивайте и не устанавливайте из Google Play

В чём разница между прокси и глобальным режимом?

  • Примечание: Правила/Глобально относятся к правилам для трафика, проходящего через прокси-софt (исходящий трафик).
  • Правила: распределяет трафик в соответствии с правилами маршрутизации.
  • Глобально: весь трафик перенаправляется/проксируется через текущий выбор.

В чём разница между системным прокси и TUN?

  • Примечание: Системный прокси и TUN - это оба способа входа в прокси-программное обеспечение, они не имеют никакого отношения к правилам/глобальному режиму
  • Примечание: Системный прокси: только для системных настроек прокси на ПК устройствах
  • Системный прокси относится к настройкам прокси в вашей системе, что требует адаптации других программ. Программы без адаптации не будут использовать системный прокси
  • TUN: виртуальная сетевая карта, прокси-программа создаёт виртуальную сетевую карту через драйвер, и практически все запросы программ в системе будут автоматически перенаправлены на виртуальную сетевую карту (разные операционные системы немного отличаются, например, UWP-приложения в Windows), без необходимости адаптации других программ

Список проблем при работе

Примечание: Если возникла проблема, сначала обновите до последней версии

Ошибка добавления конфигурации/обновления конфигурации

  • Connection reset by peer или Connection refused: Чаще всего причина в том, что запрос подписки заблокирован, попробуйте снова после открытия VPN/прокси
  • http statusCode:404: Подписка не существует, попробуйте снова после изменения ссылки подписки или изменения UserAgent (провайдер может отправлять разные типы конфигураций подписки на основе UserAgent)
  • http statusCode:403: Запрос отклонён провайдером, попробуйте другую ссылку подписки
  • Failed host lookup: Разрешение доменного имени ссылки подписки не удалось, попробуйте снова после открытия VPN/прокси
  • http response is empty: Пустой ответ провайдера, попробуйте другую ссылку подписки или измените UserAgent
  • No server available: Если вы можете добавить её с помощью других приложений, попробуйте изменить UserAgent и повторите попытку

Как проверить, поддерживает ли моя текущая сеть IPv6?

  • Отключите Karing
  • Нажмите на тест скорости на главном экране (это должен быть тест скорости, который идет с приложением. Если вы установили пользовательский тест скорости, обратитесь к Google за использованием)
  • Найдите страницу с надписью "Подключено через XXX", если XXX - это IPv6, это означает, что ваша текущая сеть поддерживает IPv6

Почему узел CF может не подключаться должным образом в Karing?

  • Узлы CF не могут использовать DNS Cloudflare. Снимите флажок "Cloudflare DNS" в настройках DNS или переключитесь на другого провайдера DNS.

Некоторые приложения не могут подключиться или возникают ошибки подключения

  • Сначала обновите до последней версии
  • Сначала необходимо подтвердить, что используемый прокси-узел может быть подключен, и настройки страны и региона в [Перенаправлении] правильны
  • Откройте [Главный экран]-[Обнаружение сети], чтобы проверить, может ли доменное имя нормально получить доступ, и измените соответствующие настройки в соответствии с сообщением об ошибке
  • Если вы не знаете, как установить доменное имя на предыдущем шаге, вы можете перейти в [Главный экран]-[Состояние подключения], чтобы проверить, есть ли запрос соответствующего доменного имени (вам нужно сначала переключить проблемное приложение и обновить приложение)
  • Если на шаге 4 нет запроса связанного доменного имени, это, по сути, проблема с разрешением DNS. Перейдите в [Главный экран]-[DNS]-[Сервер]-выберите настройку сервера-определите задержку сервера-установите соответствующий DNS-сервер в соответствии с задержкой
  • Если на шаге 4 есть запрос связанного доменного имени, проверьте, правильно ли выполнено перенаправление. При ошибке перенаправления измените соответствующие правила перенаправления

MacOS: Нажмите для подключения, и оно автоматически отключится после успешного подключения

  • Проверьте 'Консоль' - 'Отчеты о сбоях' на наличие файлов, связанных с 'karingServiceSE'. Если да, это указывает на сбой процесса VPN. Отправьте файл сбоя разработчику для анализа.
  • Перейдите в Настройки - Конфиденциальность и безопасность - Полный доступ к диску - Включите разрешения для 'karingServiceSE'.

На телефоне Android Xiaomi нельзя установить, появляется предупреждение о высокорисковом приложении

  • Эта проблема не связана с версией Karing. В настоящее время все версии Karing будут иметь эту проблему на телефонах Xiaomi, потому что Karing занесен в черный список бэкенда Xiaomi
  • Решение: Выйдите из установщика, отключите все сети (переключитесь в режим полета) и повторите попытку установки

Система Android, часто сбои при открытии соединения

  • Измените Настройки-TUN-Сетевой стек на другое значение (например, gvisor) и попробуйте снова подключиться

Ошибка Android: panic: invalid page type

  • Отключите соединение - Настройки - Очистите кэш и попробуйте снова

Система Android, ошибка при открытии соединения

  • Ошибка: android.content.ActivityNotFoundException: Unable to find explicit activity class {com.android.vpndialogs/com.android.vpndialogs.ConfirmDialog}
    Причина: Система Android удалила/упростила модуль VPN, что привело к невозможности правильного подключения VPN
    Решение: Переключитесь на другое устройство Android
  • Ошибка: process is bad
    Решение: Начните подключение из центра управления
  • Ошибка: configure tun interface: com.bly.chaos.plugin.stub.VpnServiceStub does not require android.permission.BIND_VPN_SERVICE
    Решение: Не включайте клонирование для VPN-приложений

Android отключается после подключения на некоторое время или при переходе в фоновый режим

  • Измените политику энергосбережения Karing на "Неограниченную"
  • Если ваш процессор MediaTek, вам также может потребоваться отключить переключатель оптимизации Karing в "Dura Speed"
  • Включите переключатель "Постоянный VPN" для Karing в системных настройках VPN

Пульт дистанционного управления Android TV не может переключать фокус (главный экран)

  • Настройки - Включите 'Поворачивать с экраном'

Невозможно скачать приложения из Google Play на Android

  • Смотрите решение проблемы с неудачными загрузками Google Play

Устройство не может подключиться к Интернету после отключения режима TUN на iOS/macOS

  • Из-за ограничений системы и различий в архитектуре программного обеспечения между iOS/macOS и Android/Windows, даже если режим TUN отключен, процесс сетевого расширения VPN всё равно будет запущен на iOS/macOS.
