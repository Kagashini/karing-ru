Маршрутизация трафика - одна из самых мощных возможностей Karing. С помощью правил маршрутизации вы можете точно определить, какой трафик должен идти через VPN, какой напрямую, а какой блокировать. Эта статья подробно расскажет о принципах работы маршрутизации и настройке правил.

Введение в маршрутизацию

Маршрутизация в Karing позволяет гибко управлять направлением сетевого трафика. Вместо того чтобы направлять весь трафик через VPN (как в режиме "Глобально"), вы можете создать правила, которые определяют, как обрабатывать трафик к определенным сайтам, приложениям или сетям.

Основные преимущества маршрутизации:

  • Экономия трафика VPN - только нужный трафик идет через прокси
  • Повышение скорости - локальный трафик идет напрямую
  • Обход ограничений - заблокированные сайты идут через VPN
  • Безопасность - нежелательный трафик можно блокировать
  • Гибкость - точная настройка под свои нужды

Для включения маршрутизации:

  1. Откройте главное окно Karing
  2. Убедитесь, что у вас есть активный профиль
  3. Включите переключатель "Правила" (рядом с переключателем "Глобально")

Типы правил

В Karing доступны различные типы правил маршрутизации. Каждое правило имеет тип, параметры и действие. Рассмотрим основные типы:

Domain (Домен)

Правило применяется к определенным доменам и их поддоменам. Например, google.com или *.google.com

IP

Правило применяется к определенным IP-адресам или диапазонам. Например, 8.8.8.8 или 192.168.1.0/24

Process (Процесс)

Правило применяется к трафику от определенных процессов/приложений. Например, chrome.exe или firefox

Port (Порт)

Правило применяется к трафику на определенных портах. Например, 80 (HTTP) или 443 (HTTPS)

Действия правил

Каждое правило может выполнять одно из следующих действий:

Proxy (Прокси)

Трафик идет через VPN

Direct (Напрямую)

Трафик идет напрямую

Block (Блокировать)

Трафик блокируется

Создание правил

Чтобы создать новое правило:

  1. Откройте настройки Karing
  2. Перейдите в раздел "Правила" → "Личные правила"
  3. Нажмите кнопку "+" или "Добавить правило"
  4. Выберите тип правила
  5. Введите параметры правила
  6. Выберите действие (Proxy/Direct/Block)
  7. Сохраните правило

Примеры правил

Блокировка рекламы:

Тип: Domain

Параметр: ads.google.com

Действие: Block

Обход блокировки YouTube:

Тип: Domain

Параметр: youtube.com

Действие: Proxy

Локальный трафик напрямую:

Тип: IP

Параметр: 192.168.0.0/16

Действие: Direct

Приоритет правил

Правила в Karing обрабатываются в порядке их следования в списке. Первое правило, которое подходит для данного трафика, будет применено. Поэтому порядок правил очень важен.

Рекомендуемый порядок правил (сверху вниз):

  1. Правила блокировки (самые специфичные)
  2. Правила для конкретных доменов/приложений
  3. Правила для категорий трафика
  4. Общие правила (например, GFW)
  5. Правило по умолчанию (обычно Direct)

Совет: Вы можете перетаскивать правила в списке, чтобы изменить их порядок. Правила с более высоким приоритетом должны находиться выше.

Предопределенные правила

Karing предоставляет набор предопределенных правил, которые можно легко добавить в свои настройки. Эти правила содержат списки доменов и IP-адресов для различных категорий:

GFW

Список заблокированных ресурсов в Китае. Содержит тысячи доменов и IP-адресов.

Private Network

Правила для локальных сетей (192.168.x.x, 10.x.x.x и т.д.)

Advertising

Списки рекламных доменов для блокировки

GeoIP

Правила на основе географического положения IP-адресов

Чтобы добавить предопределенное правило:

  1. Откройте настройки Karing
  2. Перейдите в раздел "Правила" → "Шаблоны личных правил"
  3. Выберите нужное правило из списка
  4. Нажмите "Добавить" или переключатель
  5. Настройте действие по умолчанию (Proxy/Direct/Block)

Расширенные возможности

Группы правил

Вы можете объединять правила в группы для удобного управления. Это особенно полезно при большом количестве правил.

Чтобы создать группу правил:

  1. В разделе "Личные правила" нажмите "Добавить группу"
  2. Введите название группы
  3. Добавьте правила в группу
  4. Группу можно включать/отключать целиком

Импорт/экспорт правил

Вы можете импортировать и экспортировать правила в формате JSON. Это удобно для резервного копирования или переноса настроек между устройствами.

Для экспорта правил:

  1. Откройте настройки Karing
  2. Перейдите в раздел "Резервное копирование и синхронизация"
  3. Выберите экспорт только правил

Для импорта правил:

  1. Откройте настройки Karing
  2. Перейдите в раздел "Резервное копирование и синхронизация"
  3. Выберите импорт из файла
  4. Выберите нужные компоненты для импорта
