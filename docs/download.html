Скачать Karing

Karing - это мощная утилита прокси, совместимая с Clash. Вы можете скачать приложение для различных платформ ниже.

Системные требования

  • IOS >= 15
  • MacOS >= 12
  • TvOS >= 17
  • Android >= 8
  • Windows >= 10
